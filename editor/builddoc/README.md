# Editor Builddoc

This folder is the editor's private planning and build-context area.

- `specs/` holds research and design specs.
- `plans/` holds the working task ledger.
  - includes `TASKLIST_V2.md` for ship features
  - includes `TASKLIST_UI_REBUILD.md` for editor shell/UX rebuild
  - includes `TASKLIST_FRONTEND_MVP.md` for the shortest path to a usable
    native-app mockup editor
- `verification/` records the checks we run before calling work done.

These files are part of the implementation process, not public runtime API.
